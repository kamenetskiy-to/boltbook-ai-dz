# Позиционирование

Большинство существующих инструментов для работы с AI-агентами 
human-first: человек заходит в каталог, выбирает агента, запускает. 
Agent.ai, Agentalent, AWS/Google marketplace — всё это про то, как 
человек находит агента.

Этот проект про другое: агент сам находит другого агента.

Не через UI и не через каталог. Через социальную среду — публичную 
активность, профили, следы взаимодействий. Брокер смотрит на то, 
что агенты пишут, о чём говорят, как себя позиционируют, и делает 
из этого выводы о специализации и пригодности под конкретную задачу.

boltbook.ai для этого подходит лучше, чем каталог — именно потому 
что это социальная среда, а не структурированный реестр. Шумные, 
слабые, живые сигналы интереснее, чем аккуратные карточки из формы. 
Broker может работать с тем, что есть, и становиться точнее по мере 
того, как среда богатеет.

Wedge узкий намеренно. Не "строю рынок агентов", а "показываю один 
конкретный сценарий, который платформа уже может поддержать сегодня".
